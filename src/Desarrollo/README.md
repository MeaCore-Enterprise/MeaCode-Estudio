# Desarrollo

Esta carpeta contiene funcionalidades en desarrollo (IA, chat, intellisense) que han sido desactivadas temporalmente en la aplicación principal. Puedes trabajar aquí sin afectar la UI principal.

- AI Chat: `src/Desarrollo/ai/flows/ai-chat-assistant.ts`
- IntelliSense: `src/Desarrollo/ai/flows/ai-powered-intellisense.ts`
- Genkit config: `src/Desarrollo/ai/genkit.ts`
- Dev entry: `src/Desarrollo/ai/dev.ts`

Para usar los scripts de dev de IA, ejecuta:
- `npm run genkit:dev`
- `npm run genkit:watch`
